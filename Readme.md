# 📊 Adult Income Census Dashboard | Power BI

## 📌 Project Overview

The Adult Income Census Dashboard is an interactive Power BI project developed to analyze demographic and income patterns using the Adult Census Income dataset.

The dashboard enables users to explore how factors such as education, occupation, gender, marital status, age, and working hours influence income levels through dynamic KPIs, interactive visualizations, and automated business insights.

---

## 🎯 Objectives

- Analyze income distribution across different demographics.
- Compare High Income (>50K) and Low Income (<=50K) populations.
- Create an interactive dashboard for business users.
- Generate dynamic insights using DAX.
- Build an end-to-end Power BI reporting solution.

## 📈 Dashboard Features

### Dynamic KPIs

- Average Age
- Total Individuals
- High Income Percentage (>50K)
- Low Income Percentage (<=50K)

All KPIs automatically update based on slicer selections.

---

### Interactive Filters

- Gender
- Relationship
- Occupation

All visuals, KPIs, and insights respond dynamically to filter selections.

---

### Visualizations

- Income Distribution by Education
- Income by Gender
- Marital Status vs Income
- Income by Hours Worked
- Top 5 Occupations by Population
- Age Group vs Income

---

### Dynamic Business Insights

One of the key features of this dashboard is the Dynamic Insights Panel.

Insights are generated using DAX measures and automatically change based on user-selected filters.

Example:

> Only 24% of individuals earn above $50K while 76% earn $50K or less.

This provides users with real-time analytical observations.

---

## 💡 Key Insights

- Majority of individuals earn <=50K.
- Higher education levels generally have a greater proportion of high-income individuals.
- Income distribution varies significantly across occupations.
- Working more than 40 hours per week is common among higher-income groups.
- Income trends differ across gender and marital status.
